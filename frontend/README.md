## Frontend - Resumo com IA

Interface web desenvolvida em **React** para interação com a API de processamento de texto com IA.

---

## Tecnologias

- React
- JavaScript
- Fetch API

---

## Funcionalidade

O usuário pode:

1. Inserir um texto
2. Enviar para a API
3. Receber um resumo gerado por IA
